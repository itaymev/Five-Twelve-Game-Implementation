# Five-Twelve-Game-Implementation
Spin on the classic 2048 game, built in class and intended for practice and learning uses only.

Please read the README in /FiveTwelve-master/ before running this code. Run game_manager.py and use WASD to play the game. For this implementation to run properly, please ensure you have the latest version of python installed and the lastest version of python turtles installed.

Thank you!
